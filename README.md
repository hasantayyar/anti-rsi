======================
Anti Rsi Notifier
======================

Usage : 

    $ antirsi start
    $ antirsi stop

![screenshot](https://raw.github.com/hasantayyar/anti-rsi/master/preview.jpg)



Installation
------------


    $ python setup.py install


Unit Tests
----------

Unit tests are run from the ``setup.py`` target ``test``::

    $ python setup.py test
